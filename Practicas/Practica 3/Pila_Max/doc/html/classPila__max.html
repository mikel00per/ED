Descripción detallada

template<class T>
class Pila_max< T >

T.D.A. List.

T.D.A. Pila.

La implementación interna se basa en un vector dinámico de donde se insertan y sacan datos a modo de cola LIFO, Last Input First Outout, el primero que entra es el primero que sale. Es la misma idea que el funcionamiento de la pila con las llamadas de las funciones.
